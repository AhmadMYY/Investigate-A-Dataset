Introduction¶

 - This project is about investigating an European soccer database that is in sqllite containing many tables like: Country', 'League', 'Player', 'Player Attributes', 'Team', 'Team Attributes' and 'Match', the database contains information about 11 european leagues between 2008 and 2016 and I shall practice what I have learnt about data wrangling, analysis then come up with conclusions and answers to questions related to this database using visulatizations with matplotlib 
 The Questions I am going to ask are as follows:
    1- What teams improved the most over the time period?

    2- Which players had the most penalties?

    3- What team attributes lead to the most victories ?

    4- What player had the highest overall_rating in 2013?

    5- Which teams won the most games in the top 3 leagues in Europe?

    6-How much does the player attributes affect the overall rating?

    So Let's dig in!!

#Adding a column for match winner
def winner(row):
    home_score = row[0]
    away_score = row[1]
    home_team_name = row[2]
    away_team_name = row[3]
    
    if home_score > away_score:
        return home_team_name
    elif home_score < away_score:
        return away_team_name
    else:
        return 'Draw'
match_df['winner']  = match_df[['home_team_goal', 'away_team_goal', 'home_team_name', 'away_team_name']].apply(winner, axis=1)

#Adding a column for points
def points(row):
    home_score = row[0]
    away_score = row[1]
    home_team_name = row[2]
    away_team_name = row[3]
    
    if home_score > away_score:
        return ('{} gained 3 points'.format(home_team_name))
    elif home_score < away_score:
        return ('{} gained 3 points'.format(away_team_name))
    else:
        return ('{} and {} gained 1 point'.format(home_team_name, away_team_name))
match_df['points_indicator']  = match_df[['home_team_goal', 'away_team_goal', 'home_team_name', 'away_team_name']].apply(points, axis=1)
